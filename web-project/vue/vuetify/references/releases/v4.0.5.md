# v4.0.5

## Notable Fixes

- SSR: avoids errors when rendering with latest Vue.
- Types: supports `noUncheckedSideEffectImports`.
- `useActivator`: avoids closing a hovered menu when a tooltip hides.
- `VSnackbar`: avoids blocking navigation.

## Labs

- `VPie`: hovered slices shrink back on mouseleave.
