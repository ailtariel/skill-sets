# v4.0.2

## Features

- Icons: added Material Symbols iconset via UnoCSS.
- `VDataTable`: bottom slot exposes `prevPage`, `nextPage`, and `setPage`.

## Notable Fixes

- Hotkey delimiter aliases.
- Missing `rounded="md"` size.
- Vue Router v5 compatibility by replacing deprecated `next()`.
- `VDataTable` respects `disableSort` for sortable headers.
- `VNumberInput` prevents input changes when readonly.
- `VSelect` / `VAutocomplete` / `VCombobox` allow focus to leave to other fields.

## Labs

- `VAvatarGroup`: `limit` behavior changed to cover the overflow item.
- `VCommandPalette`: added `closeOnSelect` prop and `before-select` event.
