# v4.0.1

Patch release with small feature additions and fixes after v4.0.0.

## Features

- Icons: added more iconsets based on UnoCSS.
- Styles: added CSS variables for fonts.

## Notable Fixes

- Theme stylesheet placement with unhead.
- `VDataTable` sorting improvements in mobile mode.
- `VGrid` gap x/y mapping.
- `VSelect` screen reader navigation.
- `VSlideGroup` avoids DOM attribute reads in computed state.
- `VSnackbar` transparent variant background.

## Labs

- `VFileUpload`: `hide-browse` hides divider.
- `VFileUpload`: exposes `controlRef` for internal `<input>`.
- `VFileUpload`: appends instead of replacing files when `multiple`.
- `VFileUpload`: integrated `VInput` and split internal logic.
