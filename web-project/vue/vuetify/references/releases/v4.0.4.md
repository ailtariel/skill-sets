# v4.0.4

## Notable Fixes

- `VCol`: corrected types for `offset-*` props.
- `VField`: label transition on pages using zoom.
- `VSelect`: prevents brief error state when clicking a menu item.
- `VTimePicker`: avoids loading all CSS utilities.

## Labs

- Rules: improved type resolution for custom rules.
- `VFileUpload`: improved adding/replacing files with dropzone click.
