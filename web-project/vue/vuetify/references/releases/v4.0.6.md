# v4.0.6

## Notable Fixes

- `VBadge`: does not accept clicks within disabled elements.
- `VField`: passes color to `icon-color` if `glow` prop is true.
- `VSelectionControl`: readonly no longer suppresses `focus-visible`.

## Labs

- `VVideo`: avoids tooltip obstruction of volume slider.
- `VVideoControls`: can be used as a standalone component.
- `VVideo`: shows progress bar by default.
