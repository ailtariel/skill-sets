# v4.1.0

Vuetify 4.1.0 (Ascendant) is the latest stable v4 release confirmed by the official Vuetify installation page and release metadata.

## Component Promotions

The following components/features were promoted from labs into the core framework:

- validation rules
- `VIconBtn`
- `VStepperVertical`
- `VPullToRefresh`
- `VFileUpload`
- `VDateInput`
- `VColorInput`
- `VPicker`

## Features

- `VCalendar`: added `interval-highlight` prop and 24-hour support.
- `VExpansionPanelTitle`: added `hover` prop to disable hover effect.
- `VSwitch`: added `size` prop, square `inset` variant, and MD3 alignment updates.
- `VDataTable`: added sortable header `aria-sort`, selection aria labels, object `loading` with `side`, `expanded` slot transition support, filtered `itemsLength` slot prop, mobile header slot, group-by `v-model:opened`, and search match highlighting.
- `VRating`: exposes `activeColor` in item slot props.
- `VTable`: added `caption` slot, aria passthrough to `<table>`, and `gridlines` prop.
- Elevation: added `hover-elevation` prop and CSS utilities.
- Rounded: accepts arbitrary values.
- Theme: added optional page transitions.
- `VCommandPalette` / `VOverlay`: introduced viewport location strategy.
- `VColorPicker`: added `hide-input-labels` prop.
- `VFileInput` / `VFileUpload`: support pasting files and folders.
- `VNumberInput`: added `grouping` prop.
- `VOtpInput`: added groups, accessibility improvements, and emoji support.
- `VPagination` / `VDataTableFooter`: can hide the last page.
- `VProgressLinear`: added `split` variant.
- `VSparkline`: added markers and tooltips.
- `VTooltip`: added `color` prop and `target="cursor"` support for hoverable tooltips.
- `VTreeview`: binds `indentLines` to item slot.

## Labs

- Added `VDateRangePicker`.
- Added `VHeatmap`.
- Added `VHighlight`.
- Added `VMonthPicker`.
- `VFileUpload`: added `loading` prop and fixes for item keys and `readonly` dropzone interactions.

## Implementation Notes

- In v4.1+ projects, prefer promoted core components before reaching for local wrappers: `VIconBtn`, `VStepperVertical`, `VPullToRefresh`, `VFileUpload`, `VDateInput`, `VColorInput`, and `VPicker`.
- For data-heavy tables, prefer the v4.1 `VDataTable` and `VTable` accessibility/slot additions before replacing tables with custom markup.
- For overlay positioning issues, re-check `VOverlay` and `VCommandPalette` viewport location strategy before adding custom positioning logic.

## Notable Fixes

- `VDialog`: fullscreen dialogs no longer apply viewport margin.
- `VDialog` / `VSnackbar`: corrected position with static location strategy.
- `VExpansionPanel`: improved focus styles.
- `VOverlay`: fixes for contained/static strategy and `location` / `origin`.
- `VSnackbarQueue`: avoids stale z-index.
