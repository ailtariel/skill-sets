# v4.0.3

## Notable Fixes

- Hotkey sequence parsing is more resilient.
- `VDataTable` reactive items from `expanded` with `return-object`.
- `VGrid` restored `no-gutters` instead of mapping it to compact density.
- `VTooltip` corrected selector for non-interactive tooltips.

## Labs

- `VMaskInput`: accepts escaped characters in masks.
- `VProgress`: added new labs component.
- `VVideo`: added `hide-progress-bar` prop.
- `VVideo`: added `src-object` prop for MediaStream/WebRTC.
