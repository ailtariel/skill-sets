# v4.0.8

Patch release with focus, overlay, select/menu, theme, and form-performance fixes.

## Bug Fixes

- Theme: supports transparent hex colors for CSS variables.
- `VAutocomplete` / `VCombobox`: clicking empty space no longer closes the menu.
- `VBtnToggle`: restores active overlay when no color is set.
- `VOverlay`: restores focus without preventing Enter keydown.
- `VOverlay`: pulls focus back to the trap when it briefly escapes.
- `VSelect` / `VAutocomplete` / `VCombobox`: keeps menu open when scrolling.
- `VSwitch`: suppresses user-agent styles to avoid click dead zones.

## Performance

- `VForm`: avoids eager `.value` reads in default slot props.
